# E-commerce Orders & Analytics API

A RESTful E-commerce backend built with Node.js and Express.js featuring Orders lifecycle management, Product stock management, and Analytics computation using Higher-Order JavaScript functions.

## Features

✅ **Orders Lifecycle Management**: placed → shipped → delivered
✅ **Product Stock Management**: Real-time stock updates
✅ **Analytics Computation**: Using Higher-Order Functions (map, forEach, filter, reduce)
✅ **Revenue Calculation**: Mandatory revenue formula implementation
✅ **Soft Delete**: Order cancellation without hard delete
✅ **Data Persistence**: Local db.json file
✅ **Clean Architecture**: Separate routes for products, orders, and analytics
